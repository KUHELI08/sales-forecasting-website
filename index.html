About The Project




This project implements a time-series forecasting model using LSTM and GRU neural networks to predict future sales trends. By training on historical sales data, the model helps businesses with demand planning and decision making.

Features include:
- Sequential learning with LSTM and GRU
- Sliding window technique for forecasting
- MinMax Scaling for normalization
- Visualization of actual vs predicted sales
